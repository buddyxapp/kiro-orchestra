Features

⚡
Parallel Execution
Multiple workers run tasks simultaneously
🧠
Event-Driven Master
Stays responsive while workers are busy. Accepts new input anytime.
📚
LLM Wiki (per agent)
Persistent knowledge that compounds over time. Long results saved to files.
🎯
@mention Routing
Direct messages to specific agents or broadcast to all
🤖
Model Selection
Each agent can use a different AI model (Opus, Sonnet, Haiku, etc.)
⏸️
Cancel Anytime
Stop individual agents or interrupt everything with one click
🔌
Full MCP Access
Every agent has Salesforce, Outlook, AWS CLI, and all configured tools
📷
Image Paste
Ctrl+V screenshots directly into chat. Agents analyze via read tool.
📊
Activity Panel
Right-side panel shows real-time tool calls per agent. Chat stays clean.
🏠
100% Local
Runs on localhost. No external services. No API keys beyond kiro-cli.
